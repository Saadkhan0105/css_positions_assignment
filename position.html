Explain the difference between Absolute and Relative positioning.

Absolute Positioning Relative Positioning
Syntax: { position: absolute; } Syntax: { position: relative; }
With relative positioning, an element can be moved up, down, left or right from its normal position. With absolute positioning, an element is positioned relative to its nearest positioned ancestor.
It will not break the format document flow to position the element on the page. It will break the format document flow to position the element on the page.
